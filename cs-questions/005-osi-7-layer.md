# CS 질문 #5: OSI 7계층 모델

## 난이도: ⭐⭐ (중간)

## 질문

네트워크 통신을 설명하는 OSI 7계층 모델에 대해 설명해주세요.

각 계층의 이름과 주요 역할을 설명해주세요.

### 힌트
- 7계층부터 1계층 순서로 (또는 1계층부터 7계층 순서로) 설명하면 좋습니다
- 각 계층이 어떤 프로토콜이나 기술을 사용하는지 예시를 들면 더 좋습니다
- "All People Seem To Need Data Processing" 같은 암기법이 있어요!

### 참고
실제 인터넷은 TCP/IP 4계층 모델을 많이 사용하지만, OSI 7계층은 네트워크를 이해하는 개념적 프레임워크로 매우 중요합니다.
