# CS 기초 질문 #1: 컴퓨터의 데이터 저장 단위 - 정답

**날짜**: 2025-11-01

## 당신의 답변

> 가장 작은 단위는 비트(bite)야 이건 0과 1로 표현하지,
> 8비트가 모이면 1바이트(byte)가돼
> 1024바이트는 1메가바이트고, 1024메가바이트는 1기가바이트가돼

## 피드백

거의 정확합니다! 다만 몇 가지 수정사항이 있습니다:

### 맞은 부분 ✅
- 가장 작은 단위는 **비트(bit)** - 0과 1로 표현
- 8비트 = 1바이트(byte)

### 수정이 필요한 부분 ⚠️

1. **철자**: "bite"가 아니라 "**bit**"입니다
   - bit = binary digit (이진수)의 줄임말

2. **단위 순서**:
   - 1024 바이트(Byte) = 1 킬로바이트(KB)
   - 1024 KB = 1 메가바이트(MB) ← 여기를 놓치셨네요!
   - 1024 MB = 1 기가바이트(GB)
   - 1024 GB = 1 테라바이트(TB)
   - 1024 TB = 1 페타바이트(PB)

## 정확한 정답

### 데이터 저장 단위 체계

**기본 단위:**
- **비트(bit)**: 컴퓨터에서 데이터를 저장하는 가장 작은 단위
  - 0 또는 1의 값을 가짐
  - Binary Digit의 줄임말

- **바이트(Byte)**: 8비트가 모인 단위
  - 1 Byte = 8 bits
  - 하나의 문자(알파벳, 숫자 등)를 표현할 수 있는 최소 단위

**더 큰 단위들:**
```
1 Byte = 8 bits
1 KB (킬로바이트) = 1024 Bytes
1 MB (메가바이트) = 1024 KB
1 GB (기가바이트) = 1024 MB
1 TB (테라바이트) = 1024 GB
1 PB (페타바이트) = 1024 TB
```

### 왜 1024인가?

컴퓨터는 이진법(Binary)을 사용하기 때문입니다.
- 2^10 = 1024
- 1000이 아니라 1024를 사용하는 이유는 2의 거듭제곱으로 계산하기 때문

### 실생활 예시

- **1 Byte**: 영문자 1개 (예: 'A')
- **1 KB**: 짧은 텍스트 문서
- **1 MB**: 고화질 사진 1장, MP3 음악 1분
- **1 GB**: 영화 1편 (SD 화질), 음악 약 250곡
- **1 TB**: 영화 약 500편, 사진 약 25만장

## 추가 학습 포인트

### 비트와 바이트 표기법
- **소문자 b** = bit (비트)
- **대문자 B** = Byte (바이트)
- 예: 100Mbps (메가비트/초) vs 100MB (메가바이트)
- 인터넷 속도는 주로 **bps**(bits per second)로 표기

### 실제 저장 용량의 차이
하드디스크를 사면 표기된 용량보다 실제 사용 가능한 용량이 작은 이유:
- 제조사는 1KB = 1000Bytes로 계산
- 운영체제는 1KB = 1024Bytes로 계산
- 이 차이 때문에 실제 용량이 적게 보임
